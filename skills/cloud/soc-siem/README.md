# soc-siem

> Build and run security monitoring (SOC / SIEM) for cloud and VMs on an open-source stack — Wazuh for host IDS, FIM, and vulnerability detection; Suricata for network IDS/IPS; Grafana as the single pane. Use when setting up a SOC or SIEM, for intrusion detection, log analysis, or threat detection, writing or tuning detection rules, mapping detections to MITRE ATT&CK, cutting false positives, or wiring Wazuh, Suricata, or Grafana.
