# teach

> Explain a concept, codebase, or system at exactly the learner's level — calibrate first, build from the known, check understanding — never a jargon dump. Use when the user asks you to teach, explain, walk through, or help them understand a concept, function, module, error, or system; when they ask "how does X work", "what is X", "ELI5", or "I don't get X"; when a prior explanation missed and they ask again.

## What it does

1. Find the level.
2. Read the material.
3. Answer in one line first.
4. Build from the known with one example.
5. Check by application, not by nodding.
6. Layer or repair on the response.
