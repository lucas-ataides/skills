## What it does

1. Frame the thesis, the goal, and the constraint.
2. Diverge, then converge to a ranked few.
3. Pick the top bets and attach a success metric to each.
4. Define the funnel and instrument it from one source of truth.
5. Set targets, then find the bottleneck.
6. Stand up retention signals and route the base by health.
7. Diagnose each at-risk driver, match one play, and close the loop.
