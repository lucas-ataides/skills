# Review 2026-07-01: findings and fixes

Full-session review of everything built (plugin, toolchain, hooks, memory protocol, radar,
Linear, consolidation). Gate was green going in; these are the gaps the pass found.

## Fixed (commit 7311e59)

1. **Scaffold template lacked the second-brain protocol line** — every new skill was born
   non-compliant with the prime/feed doctrine all 43 carry. Template fixed; probe-scaffold
   verified lint-clean with the line present.
2. **Router had no route to `agent-orchestration`** — the merged skill was unreachable by
   nudge (fan-out/parallelize/loop prompts routed nowhere); `bootstrap` prompts likewise.
   Both routed + selftested.
3. **No drift gate** — the manifest/disk/README-count class of drift bit twice (version pin;
   dead manifest entries). Now a pytest gate: manifest ≡ disk ≡ README count, and the
   marketplace must carry no version pin.
4. **`skill-cursor` left stale rules behind** — a merged-away skill's `.mdc` would linger in
   every project (the 997-skill lesson in miniature). `--prune` deletes only files carrying
   the generated marker, through `skillkit.safe_remove`.
5. **`meta/setup` enumerated a stale CLI list** — drift-proofed to "the `skill-*` toolchain".
6. **Stale egg-info build dirs** (incl. ldatb-era) removed.
7. **Duplicate PostCompact hook** in `~/.claude/settings.json` — `obsidian-bg-agent.sh` was
   registered twice → two concurrent `--dangerously-skip-permissions` headless agents racing
   the same vault after every compaction. Deduped (backup kept).
